Some Property Guides and Informations



- cltHTMLButton -


AlpahaStyle property:
AlphaStyle has 4 types. 0-NoAlpha it basically means normal button or button without Alpha, 1-Alpha it is CSS(Cascading Style Sheet) alpha style 1, 2-Alpha2 it is CSS alpha style 2, 3-Alpha3 it is CSS alpha style 3.


Colors property:
BGColor, BorderColor, Button_Color, Button_Hover_Color and FontColor properties can be edited manually by typing html colors, but to do this easier use Colors property and it will automatically edit and change the colors.


BorderStyle and BorderWidth properties:
BordrStyle has 7 types, but it will be useless with out changing or editing the BorderWidth.


Opacity and FinishOpacty properties:
Opacity and FinishOpacity controls the strength of CSS alpha style. The value should be 0 to 100.

- cltMarquee -


Behavior property:
Behavior property has 3 types. 0-Scroll basically it scrolls, 1-Slide it slide in one direction, 2-Alteranate basically it's like a Slide property but do it in two direction.


Colors property:
BGColor and FontColor properties can be edited manually by typing html colors, but to do this easier use Colors property and it will automatically edit and change the colors.


Direction property:
Direction property has 3 direction. Left, right and up.
